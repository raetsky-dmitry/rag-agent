# Ноутбук Lenovo IdeaPad 3

ID товара: 1
Категория: Ноутбуки

## Описание

Lenovo IdeaPad 3 — надёжный ноутбук для повседневных задач: работы с документами, учёбы, просмотра видео и веб-серфинга. Сочетает доступную цену с достаточной производительностью для большинства пользователей.

## Характеристики

- Процессор: Intel Core i5-1235U (10 ядер, до 4.4 ГГц)
- Оперативная память: 8 ГБ DDR4
- Накопитель: 512 ГБ SSD (M.2 NVMe)
- Дисплей: 15.6", IPS, 1920×1080, 60 Гц, антибликовое покрытие
- Видеокарта: Intel Iris Xe Graphics (встроенная)
- Аккумулятор: 45 Вт⋅ч, до 7 часов работы
- Операционная система: Windows 11 Home
- Порты: 2× USB-A 3.0, 1× USB-C, HDMI, картридер, 3.5 мм аудио
- Вес: 1.7 кг
- Цвет: серый (Arctic Grey)

## Преимущества

Быстрая загрузка благодаря SSD-накопителю. Матовый экран комфортен при работе при ярком освещении. Клавиатура с цифровым блоком удобна для работы с таблицами. Тихая работа в режиме обычной нагрузки.
